# Sistema de Gestão de Contactos

Projeto desenvolvido na disciplina de Linguagem de Programação IV.

## Objetivo
Gerir contactos com persistência em ficheiros.

## Funcionalidades
- Adicionar contacto
- Listar contactos
- Editar contactos
- Remover contacto
- Guardar em ficheiros
- Carregar ficheiros
- Pesquisar contacto
- Sair
## Linguagem usada
Java.

O relatório está na pasta /relatorio.
